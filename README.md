# Expense Tracker Pro

A professional expense tracking web application built with Flask and SQLite.

This project is an upgraded version of my JSON-based Expense Tracker and is being developed as a portfolio project to learn real-world backend development, databases, authentication, and web application architecture.

## Features

### Current Features

* Flask web application setup
* SQLite database integration
* SQLAlchemy ORM
* Git version control

### Planned Features

* User Registration and Login
* Secure Password Hashing
* Add, Edit, and Delete Expenses
* Expense Categories
* Monthly Budget Tracking
* Expense Analytics Dashboard
* Pie Charts and Monthly Reports
* Search and Filters
* CSV Export
* REST API Endpoints

## Installation

Clone the repository:

```bash
git clone https://github.com/utsavparikh-1/expense-tracker-pro.git
```

Move into the project folder:

```bash
cd expense-tracker-pro
```

Create a virtual environment:

```bash
python -m venv venv
```

Activate the virtual environment:

Windows:

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open in browser:

```text
http://127.0.0.1:5000
```
